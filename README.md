# Evidence-Based Claim Verification System

## Overview
An evidence-based claim verification system that classifies claims as:
- Supports
- Refutes
- Not Enough Information

The system combines transformer-based semantic embeddings, TF-IDF features, handcrafted contradiction signals, and an ensemble of machine learning models.
